# Vibrant Personal Website

## Setup Instructions

1. Clone the repository
   ```bash
   git clone https://github.com/wanggaoxiong/codex-demo.git
   ```
2. Navigate to the project directory
   ```bash
   cd codex-demo
   ```
3. Install dependencies for both React and Vue
   - For React
   ```bash
   cd react-app
   npm install
   ```
   - For Vue
   ```bash
   cd vue-app
   npm install
   ```
4. Start the applications
   - For React
   ```bash
   npm run dev
   ```
   - For Vue
   ```bash
   npm run dev
   ```
